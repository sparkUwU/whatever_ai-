# whatever_ai-
learning AI by doing random easy projects XD.


Alright. Let’s **reset everything** and follow the **KISS rule (Keep It Simple, Stupid)** exactly like you asked.

Forget fancy terms. Forget “AI engineer roadmap” videos.
We will build **understanding through tiny projects**.

Think of this like **learning robotics with Lego before building a spaceship**.

Your path will be:

**Programming → Data → Machine Learning → Deep Learning → Small AI systems**

Every stage has **very small projects**.

---

Goal: **Be comfortable with Python and data.**

* Basic math with Python
* Jupyter Notebook

* Python
* Jupyter Notebook
